This is the web version of SaikPOS. This project was done using ASP.NET core Razor Pages.

To run the project:
 -Please reach out to kennedyjuma480@gmail.com for the database and default credentials.
 -Once you have the database restored in sql server, clone the repository to your computer.
 -Open the cloned project in Visual studio, build and run the project.
 
 The prerequisites for running this project are .NET 5 and above, SQL Server, Visual Studio and Windows IIS Server
